Projeto de DataScience de previsão de preços de imóveis no Rio de Janeiro para pessoas comuns, que utiliza informações de 2018 e 2019 do Airbnb para fazer um cálculo automático do preço da diária, que utiliza o Python como linguagem de programação e o SKLearn (biblioteca Python) como inteligência artificial para fazer as análises.
Projeto criado para fins de teste.
